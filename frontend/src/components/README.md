DonorLoyalty Component

Usage:

1. Import and render in your app:

```jsx
import DonorLoyalty from './components/DonorLoyalty'

function App() {
  return <DonorLoyalty />
}

export default App
```

2. TailwindCSS is already in this project; ensure global CSS includes `@tailwind base; @tailwind components; @tailwind utilities;`.

3. Data: sample donors are in `src/data/donors.js`. Replace or extend that file to add more donors.
